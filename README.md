# 🚀 YouTube Performance & Engagement Analytics Dashboard

## 📊 Project Overview

This project presents an interactive Power BI dashboard built to analyze YouTube channel performance, engagement behavior, and content trends. The goal is to provide actionable insights for content creators and marketers to optimize their strategies.

---

## 🎯 Objectives

* Analyze uploads, views, and subscriber trends
* Measure engagement efficiency
* Identify top-performing categories and channels
* Enable data-driven content decisions

---

## 🧹 Data Preparation

* Removed null and duplicate values
* Fixed corrupted text (encoding issues)
* Converted columns to numeric format
* Handled missing categorical values
* Filtered invalid records (zero subscribers)

---

## 📈 Key Metrics

* Total Uploads
* Maximum Uploads
* Average Views (Last 30 Days)
* Best View Rank

---

## 🧠 Advanced Insights

* Viewed by Subscriber (Engagement Efficiency)
* Consistency Score (Recent performance vs uploads)
* Subscriber Power (Brand strength indicator)

---

## 📊 Visualizations

* Line Chart: High-volume creators (>200K uploads)
* Radar Chart: Channel type vs rank
* Packed Bubble Chart: Category-wise subscribers
* Pie Chart: Audience distribution
* Bar Chart: Top 10 channels by engagement efficiency
* Decomposition Tree: Rank-wise category analysis
![Dashboard Preview](dashboard.jpg)
---

## 🎯 Key Insights

* High subscriber count does not always mean high engagement
* Some smaller channels outperform larger ones in efficiency
* Content consistency plays a critical role in performance
* Certain categories dominate audience attention
